Bing Image Search Abstraction Layer

How To Use

You can make a query by using the form at the bottom of the page or feeding the url a query such as...

jmunford-image-search.herokuapp.com/search/pizza

And you'll get a JSON Object with the image URL, page URL and alt text. It looks like this...

{"imageUrl":"http://farm3.staticflickr.com/2692/4478854669_eaf333b619_m.jpg",

"pageUrl":"http://flickr.com/photos/pjtampabay/4478854669",

"altText":"Pizza Coloring page | Flickr - Photo Sharing!"}

You can also see additional results by adding an offset to the API call like this...

jmunford-image-search.herokuapp.com/search/pizza/4
